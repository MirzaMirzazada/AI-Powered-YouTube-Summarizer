# AI-Powered-YouTube-Summarizer

An AI-powered application that extracts transcripts from YouTube videos, generates concise summaries, and enables question-answering using LLMs and vector search.

Built with:
- Gradio
- LangChain
- FAISS
- IBM Watsonx AI

---

## Features

- Extract transcripts from YouTube videos
- Generate AI-powered summaries
- Ask questions about video content
- Semantic search using embeddings + vector database
- Simple Gradio web interface
- Fast retrieval with FAISS vector indexing
